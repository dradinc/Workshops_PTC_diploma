2) Локальное тестирование и проверка:
	- установить Python 3.8 или позднее;
	- установить OpenServer;
	- запустить OpenServer и открыть phpMyAdmin;
	- выполнить sql запрос из файла workshops_ptc.sql;
	- открыть командную строку;
	- перейти в папку проекта;
	- запустить скрипт activate, который находится в папке venv/Scripts, для запуска виртуальной среды;
	- запустить app.py;
	- открыть браузер и перейти на 127.0.0.1:5000.
